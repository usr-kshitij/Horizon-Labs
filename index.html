About This Project

Horizon Labs – Mathematical Curve Fitting Toolkit is designed and developed by Kshitij G Yerawar as an academic project for the Mathematics Club at ADCET (Annasaheb Dange College of Engineering & Technology).

The objective of this toolkit is to help students understand and apply least-squares curve fitting methods including straight line, quadratic, and exponential models through interactive computational tools built using web technologies.
